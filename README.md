Capslock Magic 是一套**跨平台**、**跨应用**的键映射解决方案。 它将 ⇪ CapsLock（大写锁定键）改造为一个强力的功能修饰键（✱ Hyper ），还改造了 <kbd>3</kbd> <kbd>4</kbd> <kbd>;</kbd> 按键，适用各种日常业务场景。奇迹般地提高操作效率与生产力。
